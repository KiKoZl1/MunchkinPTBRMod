# Munchkin PT-BR v1.3.2

## Destaques

- Delay de 7 segundos antes da primeira exibicao da logo no menu.
- Hover/click effect na logo mantidos.
- Overlay estavel em Canvas (Unity UI).
- Traducoes carregadas via recurso embutido (`es_ES` -> PT-BR).

## Instalacao

1. Instale o `MelonLoader` no `Munchkin Digital` (IL2CPP).
2. Abra o jogo, mude o idioma para `Espanhol (es_ES)` e feche o jogo.
3. Extraia `MunchkinPTBRMod.dll` do zip.
4. Copie para `...\Munchkin Digital\Mods\`.
5. Inicie o jogo novamente.
